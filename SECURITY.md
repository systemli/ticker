# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| > 1.9.0 | :white_check_mark: |
| < 1.9.0 | :x:                |


## Reporting a Vulnerability

File a issue in this repository or report them via mail to support@systemli.org. We will take care as soon as possible to fix the issue.
